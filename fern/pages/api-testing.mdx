Import the Postman collection from the CheckoutPay docs bundle:

- `MevonPay.postman_collection.json`
- `MevonPay.postman_environment.json`

**Simplest option in Cursor:** open `mevonpay-api.http` and click **Send Request** above any endpoint.

Start with **Balance** (read-only), then VTU **getInfo** before any **buy** calls.
